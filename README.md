# Nepsis
Nepsis es una aplicación Android de autoconocimiento y bienestar emocional que combina tests diarios, evaluaciones personales y una experiencia gamificada inspirada en apps de hábito. Está desarrollada con Kotlin, Jetpack Compose y Navigation Compose para ofrecer una interfaz moderna, simple y escalable.

Tecnologías
Kotlin

Jetpack Compose

Material 3

Navigation Compose

Android Studio

Git y GitHub

Funciones actuales
Navegación funcional entre pantallas principales.

Pantalla de Inicio.

Pantalla de Test.

Pantalla de Historial.

Pantalla de Perfil.

Flujo base: Inicio -> Test -> Volver.

Bottom Navigation para Inicio, Historial y Perfil.

Estructura organizada por packages (navigation, presentation, ui, data, domain.model).

Base lista para seguir escalando la app.

Funciones pendientes
Cards reales de tests recomendados en Inicio.

Test diario visible desde Home.

Preguntas interactivas reales dentro de Test.

Selección de respuestas y cálculo de resultado.

Historial con datos simulados usando lista.

Perfil con nivel, racha y puntos reales.

Mejor estilo visual y jerarquía de UI.

Persistencia local de resultados.

Cómo ejecutar
Clona el repositorio.

Abre el proyecto en Android Studio.

Espera la sincronización de Gradle.

Ejecuta la app en un emulador o dispositivo Android.

Estado del proyecto
Este proyecto ya cuenta con la base funcional de navegación y estructura general de la aplicación. Actualmente se encuentra en fase MVP, con el flujo principal operativo y preparado para agregar contenido real, lógica de tests y persistencia.
